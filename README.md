# 🚀 AI-Powered Resume Builder & Screening System

## 📌 Overview

An intelligent full-stack platform that helps job seekers create ATS-friendly resumes and enables recruiters to efficiently screen candidates using AI-powered analysis. The system combines profile management, smart resume generation, job-specific customization, and automated resume screening into a single solution.

---

## ✨ Key Features

### 👤 Profile Management

* Comprehensive profile creation with 10+ sections
* Education, Experience, Projects, Skills, Certifications, Awards, Publications, and more
* Real-time auto-save functionality
* Profile photo upload with validation
* Dynamic coding profiles and co-curricular activity management

### 🤖 AI Resume Generator

* Generates professional resumes instantly
* Job-specific resume tailoring using AI
* AI-generated career objectives
* Intelligent skill matching from job descriptions
* Multiple resume templates:

  * Professional
  * Modern
  * Minimal
  * Technical
* PDF, Word, and Text export support

### 📊 AI Resume Screening

* Bulk resume analysis and processing
* ATS-style candidate scoring
* Job requirement matching
* Skills gap analysis
* Candidate categorization:

  * Highly Suitable
  * Moderate Fit
  * Needs Improvement
* Detailed recommendations and hiring insights
* CSV report generation

### 📈 Advanced Intelligence

* Trending 2025 skill recommendations
* Technical and soft skill extraction
* Experience and qualification analysis
* AI-driven resume improvement suggestions
* Real-time job-description based optimization
